# Infraestrutura Local

Esta pasta contém a infraestrutura local para desenvolvimento e testes.

## Serviços
- LocalStack (emula AWS)
- ory/fake-oauth2-server (simula OIDC/SSO)

## Como usar

### Subir ambiente
```sh
./up.sh
```

### Derrubar ambiente
```sh
./down.sh
```

Consulte o PLAN.md na raiz para o fluxo completo do projeto.
